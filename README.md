# rohimscvprojects
This repository contains all the machine learning projects listed in my CV. I currently have two projects, a breast cancer detection model, and a flight delay prediction mode. Below is a summary of each project, what each file is for, and any further relevant remarks.

1- Breast Cancer Tumor Detection Model
This project aims to determine whether a breast tumor is malignant or benign. There will be three files included:

- data.csv: The data used.
- code.ipynb: The Jupyter Notebook file in which the models were trained and tested, as well as the markdown comments explaining the methodology, which is also explained in the research paper.
- paper.pdf: The reseatch paper which contains our findings and methodologies.

This is a group project, so the work is collective, but I was solely responsible for the entire Neural Networks section of the coding, as well as the co-author of the paper report, particularly in the Neural Networks section.


2- Flight Delay Predicition Model
This project aims to create two models, one which predicts whether a flight will be delayed or not, and then one which predicts the delay time if applicable. There will be one folder and one file contained:

- data_collection.ipynb: This is the first code file of the project, which is meant to gather all the data from different sources and combine them into one massive dataset to be used by the model to train on, which applying all relevant data pre-processing methods in this file.
- flight_delay_model_data (Folder): This folder contains all the raw data which the aforementioned code tries to gather all together. It is split into folders based on the kind of data it is and where it was retrieved from. Which it is too intricate to explain in this text, the inner folder and file names should be sufficient to make navigation easy.

This project is currently in progress, but I have uploaded all work done on it as of June 2026 to showcase current progress since a vast portion of it is complete, but do keep in mind it is still in the works and the completed sections will go through alterations and revisions as I understand machine learning and data science better.
